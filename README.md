# 🚀 Arrays & Hashing | LeetCode Pattern Repository

Welcome to my **Arrays & Hashing** repository, the first module of my **45-Day Pattern-Based DSA Preparation Journey**.

This repository focuses on mastering the **Arrays & Hashing** pattern through structured learning, optimized solutions, and detailed notes. The objective is to build strong pattern recognition rather than simply increasing the number of solved problems.

---

## 🎯 Repository Goals

- Master the Arrays & Hashing pattern
- Understand when and how to apply each technique
- Write clean and optimized solutions
- Analyze Time & Space Complexity
- Build interview-ready problem-solving skills
- Maintain concise revision notes for placements

---

# 📚 Patterns Covered

## 1. Basic Array Traversal ⭐⭐⭐⭐⭐
**Pattern:** Single pass, running sum, maximum, minimum

**Representative Problems**
- Maximum Subarray
- Best Time to Buy and Sell Stock
- Max Consecutive Ones
- Find Highest Altitude

---

## 2. Frequency Counting (HashMap) ⭐⭐⭐⭐⭐
**Pattern:** Count occurrences efficiently

**Representative Problems**
- Valid Anagram
- Majority Element
- Group Anagrams
- First Unique Character

---

## 3. HashSet Lookup ⭐⭐⭐⭐⭐
**Pattern:** Fast existence checking

**Representative Problems**
- Two Sum
- Contains Duplicate
- Missing Number
- Longest Consecutive Sequence

---

## 4. Prefix Sum ⭐⭐⭐⭐☆
**Pattern:** Running cumulative sum

**Representative Problems**
- Running Sum
- Pivot Index
- Range Sum Query

---

## 5. Prefix Sum + HashMap ⭐⭐⭐⭐⭐
**Pattern:** Efficient subarray counting/search

**Representative Problems**
- Subarray Sum Equals K
- Continuous Subarray Sum
- Contiguous Array

---

## 6. In-place Array Manipulation ⭐⭐⭐⭐⭐
**Pattern:** Modify arrays without extra space

**Representative Problems**
- Move Zeroes
- Remove Duplicates from Sorted Array
- Rotate Array

---

## 7. Sorting-Based Arrays ⭐⭐⭐⭐☆
**Pattern:** Sort first, then process

**Representative Problems**
- Minimum Absolute Difference
- Smallest Value of Rearranged Number
- Merge Intervals

---

## 8. Greedy Arrays ⭐⭐⭐⭐☆
**Pattern:** Local optimal decisions

**Representative Problems**
- Jump Game
- Best Time to Buy and Sell Stock

---

## 9. Cyclic Sort ⭐⭐⭐⭐☆
**Pattern:** Numbers in the range 0...n or 1...n

**Representative Problems**
- Missing Number
- First Missing Positive
- Find All Missing Numbers

---

## 10. Matrix (2D Arrays) ⭐⭐⭐☆
**Pattern:** Matrix traversal and manipulation

**Representative Problems**
- Spiral Matrix
- Rotate Image
- Set Matrix Zeroes

---

## 11. Bit Manipulation ⭐⭐⭐
**Pattern:** XOR-based techniques

**Representative Problems**
- Single Number

---

## 12. Simulation ⭐⭐⭐
**Pattern:** Simulate operations step by step

**Representative Problems**
- Fizz Buzz
- String Compression
- Count Subarrays of Length Three With a Condition
